Hospedagem Github

Para criar um site no Github.com devemos ter uma conta valida e um repositorio com um arquivo index.html.

Hospendando um site

  1. Criar um repositorio;
  2. Publicar o repositorio ultilizando o publish
  3. Abrir o repositorio no VSCode;
  4. Criar o index.html
  5. Cria versão usando comit
  6. Emviar para o repositorio remoto usanso push
  7. Entrar no Github.com ,acessar o repositorio ->Senttings=>Pages=>Escolher a branch do soucer
  8. Clicar na URL gerada
